# ReleascenFR

ReleascenFR est un projet open-source d'analyse statistique et de visualisation de données graphiques.

Observabilité: https://dmachard.github.io/ReleascenFR/


## Pour lancer le site en local avec docker

```bash
docker compose -f docker-compose.dev.yml up -d --build
```

Le site sera disponible à l'adresse http://localhost:8080
